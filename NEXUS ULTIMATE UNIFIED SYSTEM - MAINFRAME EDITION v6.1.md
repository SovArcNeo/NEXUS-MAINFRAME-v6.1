## ü§ñ Machine Learning Features

NEXUS v6.1 isn't just a script; it learns.

Success Prediction: Before running an agent, the system calculates a confidence score based on historical data, time of day, and system load.

Self-Correction: If an agent fails, the ML model adjusts its weights to prevent future failures (Adaptive Learning Rate).

Threat Intelligence: The DramaticAnomalyDetector builds a baseline of your system's "normal" behavior and flags deviations as LOW, MEDIUM, or CRITICAL threats.

Note: ML Data is stored in ~/.nexus/ml_models/ and persists between reboots.

## üîß Directory Structure

NEXUS automatically manages its own file system. By default, it creates a hidden .nexus directory in your home folder:

~/.nexus/phase_xii/ - Autonomous repair plans and synergy maps.

~/.nexus/ml_metrics.json - The brain of the system (Weights & Biases).

~/.nexus/config.json - System preferences.
